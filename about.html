ACERCA DE

El propósito de este proyecto es crear un sitio web informativo tipo diario digital llamado EINFORMATIVO, donde se publicarán las últimas noticias de interés local, nacional e internacional. La idea principal es simular el trabajo de reporteros, proporcionando a los usuarios información clara, precisa y actualizada.

Este proyecto fue desarrollado como parte de una práctica grupal para aplicar conocimientos de desarrollo web básico y el uso colaborativo de herramientas como Git y GitHub.
